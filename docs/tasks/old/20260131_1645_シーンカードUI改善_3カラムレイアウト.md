# シーンカードUI改善: 3カラムレイアウト

## 目的

ショート動画生成中の利用者が、台本の内容（特にセリフ）を素早く把握できるようにするため。

## 背景・課題

現在の`SceneCard`コンポーネントには以下の問題がある：

1. **セリフ（voiceText）が隠れている**: 台本で最も重要なセリフが折りたたみ内に格納されており、クリックしないと確認できない
2. **summaryが主役になっている**: 「シーンの説明」であるsummaryが常時表示され、実際のセリフより目立っている
3. **縦長になりがち**: 全情報を展開すると縦に長くなり、一覧性が損なわれる

## 設計

### 対象ファイル

- `apps/webapp/src/components/features/shorts-gen/script/scene-card.tsx`

### UIレイアウト

3カラムの横並びレイアウトを採用し、セリフ・字幕・概要を一目で確認できるようにする。

```
┌────────────────────────────────────────────────────────────────────────────────┐
│ Scene 1 [画像生成]                                                    [編集]  │
├──────────────────────────┬──────────────────────────┬──────────────────────────┤
│ 📢 セリフ                │ 💬 字幕                  │ 📝 概要                  │
│                          │                          │                          │
│ 家族が増えるのは         │ 1. 家族が増える喜び      │ 家族が増える喜びと       │
│ 嬉しいけど、お金の       │ 2. でも経済負担も...     │ ともに経済負担も増える   │
│ ことも考えないとね...    │                          │ 様子を描写               │
└──────────────────────────┴──────────────────────────┴──────────────────────────┘
```

### ジャンプ率（文字サイズの強弱）

視覚的な階層を作り、重要な情報を目立たせる：

| 要素 | サイズ | 色 | 備考 |
|------|--------|-----|------|
| セリフ（voiceText） | `text-sm` | 通常色 | 最も重要、読みやすく |
| 字幕（subtitles） | `text-sm` | 通常色 | セリフと同等 |
| 概要（summary） | `text-xs` | `text-muted-foreground` | 補助情報として薄め |
| Scene番号・バッジ | `text-xs` / `text-sm` | `text-muted-foreground` | ヘッダー情報 |
| ラベル（📢セリフ等） | `text-xs` | `text-muted-foreground` | カラムのラベル |

### 折りたたみの廃止

- 現在の展開/折りたたみ機能を削除
- 全情報を常時表示
- 3カラムレイアウトにより縦方向のスペースを節約

### 実装方針

1. `CardContent`をCSS Grid（`grid-cols-3`）で3分割
2. 各カラムに「ラベル + コンテンツ」を配置
3. セリフがない場合は「（セリフなし）」等のプレースホルダを表示
4. 字幕がない場合も同様
5. レスポンシブ対応: モバイルでは1カラムにフォールバック（`md:grid-cols-3`）
