# トップページ改善設計

## 目的

チームみらいの支持者（ITリテラシーが必ずしも高くない層を含む）が、clip-video機能を使って切り抜き動画を簡単に作成・SNS投稿できるようにするため、トップページおよび関連ページの情報設計を最適化する。

## 背景

- 選挙前日のタイミングで、サポーターにSNS投稿を促してムーブメントを起こしたい
- 現在のトップページは開発者視点の情報設計になっており、一般サポーターが「何をすればいいか」が伝わりにくい
- AI字幕自動生成というコア機能がトップページで一切案内されていない

## 現状の問題点

### 1. 「かんたんな使い方」にAI字幕ステップが抜けている

**対象**: `apps/webapp/src/app/page.tsx` L77-81

現在のステップ:
1. クリップ一覧を見る
2. Google Driveリンクからダウンロード
3. SNSに投稿

実際にユーザーに案内したいフロー:
1. クリップ一覧を見る
2. **気に入ったクリップに字幕をつける（AI自動生成）**
3. 完成した動画をダウンロード
4. SNSに投稿

AI字幕がこのツールの最大の差別化要素だが、トップページで一切触れられていない。

### 2. CTAがファーストビューに入らない

**対象**: `apps/webapp/src/app/page.tsx` 全体構成

現在の構成順:
1. ヒーロー
2. 背景説明（感謝と活動紹介）
3. 「切り抜き動画とは？」カード
4. 「SNSへの投稿のお願い」カード
5. **ようやく「かんたんな使い方」＋ CTAボタン** ← スクロール必須

ITリテラシーが高くないユーザーほど、スクロールせずに離脱する傾向がある。

### 3. ヒーローセクションが弱い

**対象**: `apps/webapp/src/app/page.tsx` L10-13

現在: 「みらい動画スタジオ」「はじめての方も大歓迎！」
→ ツール名が主役で、ユーザーに「何をしてほしいか」が伝わらない。

### 4. 「自分で切り抜き箇所を選びたい方へ」セクションが情報過多

**対象**: `apps/webapp/src/app/page.tsx` L92-113

「処理を開始」「文字起こし」など技術用語が含まれ、一般サポーターには難しい。選挙前日の文脈では、まずはクリップ一覧から選ぶフローに集中させたい。

### 5. ヘッダーナビの文言が技術的

**対象**: `apps/webapp/src/app/layout.tsx` L20-25

「動画一覧」「クリップ一覧」は開発者的な表現で、初見ユーザーには区別がつかない。

### 6. クリップ一覧ページの説明文が技術的

**対象**: `apps/webapp/src/app/clips/page.tsx` L24

「全動画から切り抜かれたクリップを横断的に確認できます」は技術的すぎる。

### 7. クリップ一覧テーブルに「字幕をつける」導線がない

**対象**: `apps/webapp/src/components/features/clip-list/clip-list-table.tsx`

DLボタンはあるが、「字幕をつける」ボタンがない。ユーザーはクリップタイトルをクリック→詳細ページに行く必要があるが、字幕機能の存在が分からない。

## 改善設計

### 改善A: トップページ全面改修

**対象ファイル**: `apps/webapp/src/app/page.tsx`

#### A-1. ヒーロー文言変更

変更前:
- h1: 「みらい動画スタジオ」
- サブ: 「はじめての方も大歓迎！」

変更後:
- h1: 「あなたの切り抜き動画で、チームみらいを届けよう！」
- サブ: 「かんたんステップ。スマホだけでできます。」

#### A-2. ページ構成の並び替え

変更後の構成順:
1. ヒーロー（A-1）
2. **かんたんな使い方 + CTAボタン**（A-3、最上位に移動）
3. 「切り抜き動画とは？」カード
4. 「SNSへの投稿のお願い」カード
5. 背景・感謝メッセージ（下に移動）
6. 「自分で切り抜き箇所を選びたい方へ」（A-4、折りたたみ）
7. フッターリンク（GitHubリンク含む、現状維持）

#### A-3. 「かんたんな使い方」のステップ更新

変更後:
1. 下の「切り抜き動画を探す」からクリップ一覧を見る
2. 気に入ったクリップを選んで、字幕をつける（AIが自動で考えてくれます！）
3. 完成した動画をダウンロード
4. お好きなSNS（TikTok、X、Instagramなど）に投稿！

#### A-4. 上級者セクションの折りたたみ化

HTMLの `<details>/<summary>` タグを使用してデフォルト閉じにする。shadcn/uiのAccordionは過剰なため、シンプルにdetailsタグで実装。

### 改善B: ヘッダーナビ文言改善

**対象ファイル**: `apps/webapp/src/app/layout.tsx`

変更前:
- 「動画一覧」→ `/videos`
- 「クリップ一覧」→ `/clips`

変更後:
- 「切り抜きを探す」→ `/clips`
- 「動画から作る」→ `/videos`

順番も変更: クリップ一覧（メイン導線）を左に、動画一覧（上級者向け）を右に。

### 改善C: クリップ一覧ページ改善

**対象ファイル**: `apps/webapp/src/app/clips/page.tsx`, `apps/webapp/src/components/features/clip-list/clip-list-table.tsx`

#### C-1. 説明文の変更

変更前: 「全動画から切り抜かれたクリップを横断的に確認できます」
変更後: 「気に入ったクリップを選んで、字幕をつけてSNSに投稿しよう！」

#### C-2. テーブルに「字幕をつける」ボタン追加

既存のDLボタンの隣に「字幕をつける」ボタンを追加。クリップ詳細ページ (`/clips/[id]`) へのリンク。
既存のDLボタン列と同じカラムにまとめるか、独立したアクション列とする。

各改善は対象ファイルが異なるため、並列に実装可能。

|----------|------------|----------|
