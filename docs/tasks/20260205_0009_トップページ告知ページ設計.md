# トップページ告知ページ設計

## 目的

初めてこのツールを訪れるボランティアの方が、切り抜き動画制作の目的・やり方・ツールの使い始め方を一目で理解し、すぐに作業に取りかかれるようにするため。

## 背景

- 動画一覧ページを `/videos` に移動済み
- トップページ (`/`) が空いたので、告知・説明ページを配置する
- 対象ユーザーはスマホのみの方も含む初心者ボランティア

## 変更対象ファイル

- `apps/webapp/src/app/page.tsx` — トップページの内容を告知ページに差し替え

## ページ構成

### セクション構成

以下の順に縦に並べるシンプルな1カラムレイアウトとする。

#### 1. ヒーローセクション

- タイトル: 「切り抜き動画のご協力のお願い」
- サブタイトル: 「はじめての方も大歓迎！」
- 感謝メッセージ: 「いつもあたたかいご支援をありがとうございます。」

#### 2. なぜ切り抜きが必要か

告知文のうち、以下の内容をそのまま掲載する:
- いまチームみらいでは街頭演説・演説会など日々たくさんの活動が行われている
- それらの活動はまだまだ十分に広がりきっていない
- 残り数日、SNSでの「切り抜きショート動画」の力がとても重要

#### 3. 「切り抜き動画」とは？（Card）

Cardコンポーネントを使って以下を説明:
- チームみらい公式YouTube、候補者の街頭演説や演説会イベントの動画から
- 「ここ、いいな」「伝わりやすいな」という部分（10〜60秒くらい）を切り取った動画

#### 4. 投稿のお願い

- TikTok、X、Instagramなどに投稿してほしいという呼びかけ
- 「本格的な編集は不要です」
- 「動画編集したことない」「SNSは見る専だった」という方も大歓迎

#### 5. 素材について（Card）

- 切り抜き用の素材をまとめたSlackチャンネルがある旨
- ※ユーザーからSlackチャンネルのURLやリンク先が提供されていないため、テキストのみ記載（リンクは後日追加可能な形にしておく）

#### 6. ツールを使ってみる（CTA）

- 「動画一覧を見る」ボタン → `/videos` へのリンク
- このツールで切り抜き作業ができることの簡単な説明

### スマホ対応

- 既存のTailwindレスポンシブ設定（`container mx-auto px-4`）がlayout.tsxのmainタグに適用済み
- 追加のレスポンシブ対応は不要（1カラムレイアウトのためそのまま動作する）

### 使用するコンポーネント

- `Card`, `CardHeader`, `CardTitle`, `CardContent` — 説明ブロックの囲み
- `Button` — CTAボタン
- lucide-react のアイコン（`Scissors`, `ArrowRight` など）

### 静的ページ

- データ取得なし（Server Actionやloader不要）
- `force-dynamic` 不要（静的レンダリングでOK）
